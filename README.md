# Text to PDF Website

A simple web app that converts user input text into a downloadable PDF file.

## Features
- Enter text in a textbox
- Convert text into PDF
- Download instantly

## Technologies Used
- HTML
- CSS
- JavaScript
- jsPDF Library

## How to Run
1. Clone the repository
2. Open index.html in your browser
